# PromoPageHorizontalCard

Горизонтальная карточка, занимающая всю ширину контентной области. Предназначена для эффектного представления информации на промо-страницах и в онбордингах, сочетая текстовое описание с крупным визуальным акцентом в виде изображения. Часто используется для выделения ключевых предложений, продуктов или уникальных преимуществ.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `title` | `React.ReactNode` | `'Text 2XL'` | Заголовок карточки |
| `description` | `React.ReactNode` | `'Text L'` | Описание под заголовком |
| `buttonLabel` | `React.ReactNode` | `'Text L'` | Текст кнопки действия (только для `accent`) |
| `image` | `React.ReactNode` | — | Произвольный элемент-изображение |
| `imageSrc` | `string` | — | URL изображения (приоритетнее `image`) |
| `imageAlt` | `string` | `''` | Alt-текст изображения |
| `variant` | `'default' \| 'accent'` | `'default'` | Визуальный стиль: стандартный или акцентный с кнопкой |
| `className` | `string` | `''` | Дополнительные CSS классы |

### Показ/скрытие элементов

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `hasDescription` | `boolean` | `true` | Показывает описание |
| `hasButton` | `boolean` | `true` | Показывает кнопку (только в варианте `accent`) |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `onButtonClick` | `() => void` | — | Колбэк при нажатии на кнопку |

## Адаптивность

При ширине экрана до 599px карточка переключается в адаптивную компоновку.
