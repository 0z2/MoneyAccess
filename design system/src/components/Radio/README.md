# Radio

Компонент для единичного выбора из множества.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `label` | `string` | — | Aria-метка для доступности |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isSelected` | `boolean` | `false` | Переводит кнопку в выбранное состояние |
| `isDisabled` | `boolean` | `false` | Блокирует кнопку |
| `onChange` | `() => void` | — | Колбэк при выборе |
