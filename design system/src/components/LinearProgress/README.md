# LinearProgress

Компонент для отображения информации о продолжительности и ходе процесса. Например, шаги заполнения формы, процесс загрузки или лимиты.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `variant` | `'percent' \| 'steps'` | `'percent'` | Режим отображения: непрерывный прогресс в % или дискретные шаги |
| `value` | `number` | — | Текущее значение прогресса. В режиме `percent` — от 0 до 100, в режиме `steps` — количество выполненных шагов (обязательный) |
| `maxSteps` | `number` | `5` | Общее количество шагов (только для `steps`) |
| `progressColor` | `string` | `var(--bg-brand)` | Цвет заполненной части |
| `trackColor` | `string` | `var(--container-transparent-2)` | Цвет фоновой дорожки |
| `ariaLabel` | `string` | — | Aria-метка для доступности |
| `className` | `string` | `''` | Дополнительные CSS классы |
