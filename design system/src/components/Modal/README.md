# Modal

Всплывающее окно с затемнённым оверлеем, показывающееся поверх контента страницы. Обычно используется для отображения дополнительной информации или интерактивных элементов, не требуя перехода на другую страницу.

Собирается из `ModalHeader`, зоны контента и `ModalFooter`.

---

## Modal

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `header` | `React.ReactNode` | — | Шапка окна, обычно `ModalHeader` |
| `footer` | `React.ReactNode` | — | Подвал окна, обычно `ModalFooter` |
| `children` | `React.ReactNode` | — | Прокручиваемый контент окна |
| `className` | `string` | `''` | Дополнительные CSS классы |

### Показ/скрытие элементов

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isOpen` | `boolean` | — | Открывает или скрывает окно (обязательный) |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `onClose` | `() => void` | — | Колбэк при закрытии (Escape или клик по оверлею) |
| `isOverlayCloseEnabled` | `boolean` | `true` | Разрешает закрытие по клику на оверлей |

---

## ModalHeader

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `title` | `React.ReactNode` | — | Заголовок окна |
| `leftAccessory` | `React.ReactNode` | — | Произвольный элемент слева (заменяет стрелку назад) |
| `className` | `string` | `''` | Дополнительные CSS классы |

### Показ/скрытие элементов

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `hasDefaultBackArrow` | `boolean` | `false` | Показывает стрелку «назад» слева, если не передан `leftAccessory` |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `onLeftAccessoryClick` | `() => void` | — | Колбэк при клике на левый элемент |
| `onClose` | `() => void` | — | Колбэк при клике на кнопку закрытия (×) |

---

## ModalFooter

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `layout` | `'1-button' \| '2-buttons' \| '2-horizontal-buttons' \| 'empty'` | `'1-button'` | Компоновка кнопок: одна, две вертикально, две горизонтально или пустой подвал |
| `description` | `React.ReactNode` | — | Вспомогательный текст над кнопками |
| `primaryAction` | `ModalFooterAction` | — | Основная кнопка |
| `secondaryAction` | `ModalFooterAction` | — | Вторичная кнопка (для `2-buttons` и `2-horizontal-buttons`) |
| `className` | `string` | `''` | Дополнительные CSS классы |

---

**`ModalFooterAction`:**

| Поле | Тип | Описание |
|---|---|---|
| `label` | `string` | Текст кнопки |
| `onClick` | `() => void` | Колбэк по клику |
| `isDisabled` | `boolean` | Блокирует кнопку |
| `isLoading` | `boolean` | Показывает спиннер |
| `isSelected` | `boolean` | Отображает кнопку как `primary` |
