# PageAction

Компонент предназначен для совершения действий, переходов во внутренние разделы, получения дополнительной информации.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `title` | `React.ReactNode` | — | Основной текст (обязательный) |
| `description` | `React.ReactNode` | — | Дополнительный текст под заголовком |
| `leftAccessory` | `React.ReactNode` | — | Левый аксессуар, ожидается иконка 30px |
| `variant` | `'default' \| 'danger'` | `'default'` | Визуальный стиль: стандартный или деструктивный |
| `className` | `string` | `''` | Дополнительные CSS классы |

### Показ/скрытие элементов

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `hasDescription` | `boolean` | `true` | Скрывает описание, даже если `description` передан |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `onClick` | `() => void` | — | Колбэк при нажатии |
| `isDisabled` | `boolean` | `false` | Блокирует кнопку |
