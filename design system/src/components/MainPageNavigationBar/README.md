# MainPageNavigationBar

Главная навигационная панель сайта. Содержит логотип, навигацию по разделам, информацию о клиенте и кнопки быстрых действий.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `customer` | `string` | `'ООО Ромашка'` | Название компании клиента |
| `tin` | `string` | `'ИНН 4827 1359 64'` | ИНН клиента |
| `avatarInitials` | `string` | `'НО'` | Инициалы для аватара, если не передан `avatarUrl` |
| `avatarUrl` | `string` | — | URL изображения аватара клиента |
| `logoUrl` | `string` | — | URL логотипа банка (по умолчанию — встроенный SVG) |
| `tochkaPlusUrl` | `string` | — | URL логотипа Tochka Plus |
| `activeNavItem` | `'main' \| 'payments' \| 'services'` | — | Активный пункт навигации |
| `className` | `string` | `''` | Дополнительные CSS классы |

### Показ/скрытие элементов

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `hasLive` | `boolean` | `false` | Показывает индикатор «В ЭФИРЕ» рядом с логотипом |
| `hasNewPush` | `boolean` | `true` | Показывает индикатор новых уведомлений на кнопке колокольчика |
| `hasSelect` | `boolean` | `true` | Показывает кнопку переключения клиента (шеврон) |
| `isSecondLine` | `boolean` | `false` | Показывает вторую строку с подпиской и ИНН |
| `hasSubscription` | `boolean` | `true` | Показывает логотип Tochka Plus во второй строке (требует `isSecondLine` и `tochkaPlusUrl`) |
| `hasTin` | `boolean` | `true` | Показывает ИНН во второй строке (требует `isSecondLine`) |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `onCustomerClick` | `() => void` | — | Колбэк при клике на имя клиента |
| `onNotificationsClick` | `() => void` | — | Колбэк при клике на уведомления |
| `onGiftClick` | `() => void` | — | Колбэк при клике на подарки |
| `onSettingsClick` | `() => void` | — | Колбэк при клике на настройки |
| `onLogoutClick` | `() => void` | — | Колбэк при клике на выход |
| `onNavMainClick` | `() => void` | — | Колбэк при клике на «Главная» |
| `onNavPaymentsClick` | `() => void` | — | Колбэк при клике на «Платежи» |
| `onNavServicesClick` | `() => void` | — | Колбэк при клике на «Сервисы» |

## Адаптивность

Вид выбирается автоматически через media queries:

- **до 1023px** — адаптивная версия: аватар и имя клиента слева, кнопки настроек и уведомлений справа, навигация скрыта
- **1024px – 1279px** — десктоп: логотип слева, навигация в центре, информация о клиенте и действия справа
- **1280px+** — десктоп с увеличенным пространством
