# CellLeftAccessory

Набор готовых вариантов левого аксессуара. Применяется в ячейках списков, формах и других компонентах, где нужен стандартизированный элемент слева: аватар, иконка, превью карточки или кнопка добавления.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `variant` | `'avatar' \| 'icon-30' \| 'icon-24' \| 'icon-18' \| 'card-preview' \| 'avatar-checkbox' \| 'add-button' \| 'custom'` | `'avatar'` | Вариант аксессуара |
| `content` | `React.ReactNode` | — | Произвольный контент. При наличии имеет приоритет над `variant` |
| `icon` | `React.ReactNode` | `Circle` | Иконка для вариантов `icon-*` и `add-button` |
| `avatarLabel` | `string` | `'AA'` | Инициалы для варианта `avatar` |
| `className` | `string` | `''` | Дополнительный CSS-класс |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isChecked` | `boolean` | `false` | Состояние чекбокса для варианта `avatar-checkbox` |
| `onClick` | `() => void` | — | Колбэк по клику для варианта `add-button` |
