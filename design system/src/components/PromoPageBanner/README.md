# PromoPageBanner

Крупный визуальный блок, выполняющий роль хедера (шапки) для промо-страниц, онбордингов и других контекстных страниц банка. Предназначен для создания первого яркого впечатления, донесения главного сообщения страницы и задания общего тона коммуникации.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `title` | `React.ReactNode` | `'Text 5XL'` | Заголовок в десктопной версии |
| `adaptiveTitle` | `React.ReactNode` | `'Text 3XL'` | Заголовок в адаптивной версии |
| `description` | `React.ReactNode` | `'Text XL'` | Описание в десктопной версии |
| `adaptiveDescription` | `React.ReactNode` | `'Text M'` | Описание в адаптивной версии |
| `buttonLabel` | `React.ReactNode` | `'Text L'` | Текст кнопки действия |
| `image` | `React.ReactNode` | — | Произвольный элемент-изображение |
| `imageSrc` | `string` | — | URL изображения (приоритетнее `image`) |
| `imageAlt` | `string` | `''` | Alt-текст изображения |
| `className` | `string` | `''` | Дополнительные CSS классы |

### Показ/скрытие элементов

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `hasImage` | `boolean` | `true` | Показывает блок с изображением |
| `hasDescription` | `boolean` | `true` | Показывает описание |
| `hasButton` | `boolean` | `true` | Показывает кнопку действия |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `onButtonClick` | `() => void` | — | Колбэк при нажатии на кнопку |
