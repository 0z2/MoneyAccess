# Action Form Cell

Интерактивный элемент действия для форм и списков. Используется как самостоятельный элемент или в сгруппированном стеке.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `title` | `string` | — | Основной текст ячейки |
| `description` | `string` | — | Дополнительный текст под заголовком |
| `left` | `React.ReactNode` | — | Левый аксессуар (иконка 24px) |
| `right` | `React.ReactNode` | — | Правый аксессуар (спиннер) |
| `variant` | `'single' \| 'stack-top' \| 'stack-middle' \| 'stack-bottom'` | `'single'` | Расположение ячейки: отдельная или позиция в группе |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isDisabled` | `boolean` | `false` | Блокирует ячейку |
| `onClick` | `() => void` | — | Колбэк по клику |
