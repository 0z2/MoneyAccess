# ContextMenu

Всплывающее меню с набором действий, которое появляется рядом с элементом по клику или тапу. Используется для вызова контекстных действий без перехода на другой экран.

## Props

### `ContextMenu`

#### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `trigger` | `React.ReactNode` | — | Элемент, по клику на который открывается меню |
| `items` | `ContextMenuItem[]` | — | Список действий в меню |
| `placement` | `'right' \| 'left'` | `'right'` | Сторона, в которую раскрывается меню относительно триггера |
| `className` | `string` | `''` | Дополнительный CSS-класс |

#### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isOpen` | `boolean` | — | Управляет видимостью меню |
| `onClose` | `() => void` | — | Колбэк при закрытии (выбор действия или клик вне меню) |

---

### `ContextMenuItem`

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `key` | `string` | — | Уникальный идентификатор элемента |
| `label` | `React.ReactNode` | — | Текст действия |
| `icon` | `React.ReactNode` | — | Иконка 24px слева от текста |
| `variant` | `'default' \| 'danger'` | `'default'` | Визуальный стиль: обычный или деструктивный |
| `isDisabled` | `boolean` | — | Блокирует элемент меню |
| `onClick` | `() => void` | — | Колбэк по клику на элемент |
