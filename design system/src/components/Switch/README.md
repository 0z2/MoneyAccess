# Switch

Компонент для быстрого переключения между двумя возможными состояниями.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `label` | `string` | — | Aria-метка для доступности |
| `style` | `React.CSSProperties` | — | Инлайн-стили для кастомизации |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isSelected` | `boolean` | `false` | Переводит переключатель во включённое состояние |
| `isDisabled` | `boolean` | `false` | Блокирует переключатель |
| `onChange` | `(isSelected: boolean) => void` | — | Колбэк при переключении, передаёт новое значение |
