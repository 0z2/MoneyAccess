# Chip

Компонент для одиночного или множественного выбора. Часто применяется для фильтрации контента в списках. Обычно используется в виде группы из нескольких элементов.

**Chip** — базовый вариант для выбора значения. Применяется для множественного выбора в группе.

**Chip Tab** — для переключения контента на странице. Применяется в группе с другими табами для переключения контента в рамках одной страницы.

**Chip Dropdown** — позволяет фильтровать контент, выбирая для каждого параметра значение из списка или календаря.

**Chip Action** — может вызывать действие в списке чипов. Например, показ скрытых элементов или вызов модального окна.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `children` | `React.ReactNode` | — | Текст чипа |
| `variant` | `'chip' \| 'tab' \| 'dropdown' \| 'action'` | `'chip'` | Тип чипа |
| `leftAccessory` | `'icon' \| 'logo' \| 'logo-stack'` | — | Левый аксессуар |
| `leftIcon` | `React.ReactNode` | `Circle` | Иконка для варианта `leftAccessory="icon"` |
| `badge` | `number \| string` | — | Бейдж справа (только для варианта `tab`) |
| `className` | `string` | `''` | Дополнительный CSS-класс |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isSelected` | `boolean` | `false` | Выбранное состояние |
| `isDisabled` | `boolean` | `false` | Блокирует чип |
| `isOpen` | `boolean` | `false` | Открытое состояние для варианта `dropdown` |
| `onClick` | `() => void` | — | Колбэк по клику |
| `onClose` | `(e: React.MouseEvent) => void` | — | Колбэк при удалении для варианта `action` |
