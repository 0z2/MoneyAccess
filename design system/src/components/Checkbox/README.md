# Checkbox

Элемент выбора, который позволяет отметить один или несколько вариантов из списка. Используется в формах, фильтрах и настройках. Также чекбокс может использоваться для подтверждения согласия с условиями, правилами или юридически значимыми документами.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `label` | `string` | — | Aria-метка для доступности |
| `style` | `React.CSSProperties` | — | Инлайн-стили |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isChecked` | `boolean` | `false` | Отмечен ли чекбокс |
| `isIndeterminate` | `boolean` | `false` | Промежуточное состояние (частичный выбор) |
| `isDisabled` | `boolean` | `false` | Блокирует чекбокс |
| `onChange` | `(checked: boolean) => void` | — | Колбэк при изменении состояния |
