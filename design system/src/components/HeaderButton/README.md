# HeaderButton

Кнопка или группа кнопок. Располагается в верхней части экрана под заголовком или первым блоком страницы. Используются для совершения основных или востребованных в контексте применяемого экрана действий. Если действий много, часть менее важных может сворачиваться в одну кнопку с Context Menu.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `children` | `React.ReactNode` | — | Текст кнопки (обязательный) |
| `icon` | `React.ReactNode` | — | Иконка слева от текста |
| `variant` | `'primary' \| 'secondary' \| 'danger'` | `'primary'` | Визуальный стиль: основное, второстепенное или деструктивное действие |
| `onClick` | `() => void` | — | Колбэк при нажатии |

### Показ/скрытие элементов

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `icon` | `React.ReactNode` | — | Если не передан, иконка не отображается |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isDisabled` | `boolean` | `false` | Блокирует кнопку |
| `isLoading` | `boolean` | `false` | Показывает спиннер и блокирует кнопку |
