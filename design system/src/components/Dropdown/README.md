# Dropdown

Поле выбора из раскрывающегося списка. Используется в формах, когда нужно выбрать одно значение из заранее заданного набора вариантов.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `options` | `DropdownOption[]` | — | Список вариантов для выбора |
| `placeholder` | `string` | `'Select option'` | Текст-заглушка до выбора значения |
| `label` | `string` | — | Подпись над полем |
| `description` | `string` | — | Вспомогательный текст под полем |
| `errorMessage` | `string` | — | Текст ошибки (заменяет `description` при `isError=true`) |
| `right` | `React.ReactNode` | — | Произвольный элемент справа, перед шевроном |

### Показ/скрытие элементов

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `hasChevron` | `boolean` | `true` | Показывает иконку ChevronDown справа |
| `hasHelpIcon` | `boolean` | `true` | Показывает иконку подсказки рядом с `label` |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `value` | `string` | — | Управляемое значение. Если передан — компонент переходит в controlled mode |
| `onChange` | `(value: string) => void` | — | Колбэк при выборе значения |
| `isDisabled` | `boolean` | `false` | Блокирует поле |
| `isError` | `boolean` | `false` | Переводит поле в состояние ошибки |

---

**`DropdownOption`:**

| Поле | Тип | Описание |
|---|---|---|
| `value` | `string` | Значение опции |
| `label` | `string` | Отображаемый текст |
| `isDisabled` | `boolean` | Блокирует опцию |
