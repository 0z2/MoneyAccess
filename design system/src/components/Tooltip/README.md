# Tooltip

Всплывающая подсказка, появляющаяся при наведении или фокусе на элемент-триггер. Применяется для отображения текста, содержащего дополнительную информацию.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `trigger` | `React.ReactNode` | — | Элемент-триггер, при наведении или фокусе на который появляется подсказка (обязательный) |
| `children` | `React.ReactNode` | — | Содержимое подсказки (обязательный) |
| `placement` | `'right' \| 'left'` | `'right'` | Положение подсказки относительно триггера |
| `className` | `string` | `''` | Дополнительные CSS классы для обёртки |

### Показ/скрытие элементов

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isOpen` | `boolean` | — | Контролируемое состояние видимости. Если передан — компонент переходит в controlled mode |
| `defaultOpen` | `boolean` | `false` | Начальное состояние для неконтролируемого режима |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `onOpenChange` | `(isOpen: boolean) => void` | — | Колбэк при изменении видимости |
