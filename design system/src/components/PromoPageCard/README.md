# PromoPageCard

Карточка для контентного наполнения промо-страниц, онбордингов и других контекстных страниц банка. Предназначена для структурированного и визуально привлекательного представления ключевой информации, преимуществ продукта/услуги или этапов какого-либо процесса.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `title` | `React.ReactNode` | `'Text XL'` | Заголовок карточки |
| `description` | `React.ReactNode` | `'Text M'` | Описание под заголовком |
| `avatar` | `React.ReactNode` | — | Произвольный аватар (по умолчанию — иконка с брендовым фоном) |
| `image` | `React.ReactNode` | — | Произвольный элемент-изображение |
| `imageSrc` | `string` | — | URL изображения (приоритетнее `image`) |
| `imageAlt` | `string` | `''` | Alt-текст изображения |
| `isHorizontal` | `boolean` | `false` | Горизонтальная компоновка: аватар и текст в ряд, изображение скрыто |
| `className` | `string` | `''` | Дополнительные CSS классы |

### Показ/скрытие элементов

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `hasAvatar` | `boolean` | `true` | Показывает аватар |
| `hasImage` | `boolean` | `true` | Показывает изображение |
| `hasDescription` | `boolean` | `true` | Показывает описание |
