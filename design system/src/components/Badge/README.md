# Badge

Компонент, показывающий индикатор или счётчик. Обычно используется для отображения статуса или уведомления о новых событиях. Используется поверх иконок, в списках и аксессуарах.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `value` | `number` | — | Числовое значение. Значения больше 99 отображаются как `99+` |
| `size` | `'m' \| 's' \| 'xs'` | `'m'` | Размер бейджа |
| `color` | `string` | `var(--primitive-neutral-4)` | Цвет фона |
| `textColor` | `string` | `var(--primitive-default)` | Цвет текста |
| `className` | `string` | `''` | Дополнительный CSS-класс |
