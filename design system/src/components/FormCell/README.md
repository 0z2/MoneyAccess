# FormCell

Интерактивный элемент для использования свитчера, чекбоксов и радио на странице с формой.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `title` | `string` | — | Основной текст ячейки (обязательный) |
| `subtitle` | `string` | — | Подзаголовок, отображается перед основным текстом |
| `description` | `string` | — | Дополнительное описание под основным текстом |
| `left` | `React.ReactNode` | — | Левый слот (например Avatar) |
| `right` | `React.ReactNode` | — | Правый слот для управляющего элемента (Switch, Checkbox, Radio) |
| `children` | `React.ReactNode` | — | Дополнительный контент под ячейкой |
| `variant` | `'single' \| 'stack-top' \| 'stack-middle' \| 'stack-bottom'` | `'single'` | Тип отображения ячейки: отдельная или позиция в группе |
| `className` | `string` | `''` | Дополнительные CSS классы |

### Показ/скрытие элементов

Все необязательные пропсы (`subtitle`, `description`, `left`, `right`, `children`) отображаются только при передаче значения.
