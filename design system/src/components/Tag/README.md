# Tag

Компонент, показывающий статус чего‑либо или метку.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `children` | `React.ReactNode` | — | Текст метки (обязательный) |
| `shape` | `'circle' \| 'square'` | `'square'` | Форма: скруглённая или квадратная |
| `variant` | `'filled' \| 'outlined'` | `'filled'` | Стиль: заливка или обводка |
| `size` | `'xl' \| 'l' \| 'm' \| 's'` | `'s'` | Размер метки |
| `className` | `string` | `''` | Дополнительные CSS классы |
