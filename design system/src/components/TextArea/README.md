# TextArea

Многострочное поле ввода с автоматическим расширением, поддержкой подписи, подсказки, счётчика символов и состояния ошибки.

## Props

### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `label` | `string` | — | Подпись над полем |
| `placeholder` | `string` | — | Текст-заглушка при пустом поле |
| `description` | `string` | — | Вспомогательный текст под полем |
| `errorMessage` | `string` | — | Текст ошибки (заменяет `description` при `isError=true`) |
| `maxLength` | `number` | — | Максимальное количество символов; включает счётчик над полем |

### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `value` | `string` | `''` | Управляемое значение |
| `onChange` | `(value: string) => void` | — | Колбэк при изменении значения |
| `isDisabled` | `boolean` | `false` | Блокирует поле |
| `isError` | `boolean` | `false` | Переводит поле в состояние ошибки |
