# Action Sheet

Всплывающая панель снизу экрана со списком действий, связанных с текущим контекстом. Может содержать от одного до нескольких вариантов действий. Используется для подтверждения действия или для выбора на развилке сценария.

Компонент составной: собирается из `ActionSheet`, `ActionSheetHeader`, `ActionSheetButton` и `ActionSheetFooter`.

## Props

### `ActionSheet`

#### Контент

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `header` | `React.ReactNode` | — | Шапка панели, обычно `ActionSheetHeader` |
| `children` | `React.ReactNode` | — | Список действий, обычно `ActionSheetButton` |
| `footer` | `React.ReactNode` | — | Подвал панели, обычно `ActionSheetFooter` |
| `className` | `string` | `''` | Дополнительный CSS-класс |

#### Показ/скрытие элементов

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isOverlayCloseEnabled` | `boolean` | `true` | Разрешает закрытие по клику на оверлей |

#### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isOpen` | `boolean` | — | Управляет видимостью панели |
| `onClose` | `() => void` | — | Колбэк при закрытии (клик по оверлею или Escape) |

### `ActionSheetHeader`

#### Контент

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `title` | `React.ReactNode` | — | Заголовок шапки |
| `className` | `string` | `''` | Дополнительный CSS-класс |

#### Показ/скрытие элементов

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `hasContent` | `boolean` | `true` | Показывает контент шапки. При `false` рендерится пустой разделитель |

---

### `ActionSheetButton`

#### Контент и варианты

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `title` | `React.ReactNode` | — | Текст кнопки |
| `description` | `React.ReactNode` | — | Дополнительный текст под заголовком |
| `icon` | `React.ReactNode` | — | Иконка 30px слева |
| `variant` | `'default' \| 'danger'` | `'default'` | Визуальный стиль: обычный или деструктивный |
| `className` | `string` | `''` | Дополнительный CSS-класс |

#### Показ/скрытие элементов

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `hasDescription` | `boolean` | `false` | Показывает `description` (только если он передан) |
| `hasIcon` | `boolean` | `true` | Показывает иконку (только если она передана) |

#### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isDisabled` | `boolean` | `false` | Блокирует кнопку |
| `isLoading` | `boolean` | `false` | Показывает спиннер и блокирует кнопку |
| `onClick` | `() => void` | — | Колбэк по клику |

---

### `ActionSheetFooter`

#### Состояние

| Проп | Тип | По умолчанию | Описание |
|---|---|---|---|
| `isDisabled` | `boolean` | `false` | Блокирует кнопку «Отмена» |
| `isLoading` | `boolean` | `false` | Показывает спиннер и блокирует кнопку |
| `onClick` | `() => void` | — | Колбэк по клику на «Отмена» |
| `className` | `string` | `''` | Дополнительный CSS-класс |
